# Release v0.1.0

Initial educational release.

## Included

- Screen capture backends: MSS and DXcam
- Ultralytics YOLO tracking wrapper
- HSV health/status bar classifier
- Geometry feature extraction
- Simple target selection policy
- Optional ADB input controller
- Capture benchmark, frame extraction, and training helper scripts

## Not included

- Pretrained model weights
- Dataset images
- GitHub Release asset upload, because the current connector only exposes repository file operations
